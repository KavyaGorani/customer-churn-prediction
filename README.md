# Customer Churn Prediction using Machine Learning
## Project Overview
### Customer churn refers to customers who stop using a companyâ€™s service.This project aims to predict whether a customer will churn using machine learning based on demographic, service, and billing information.The objective is to help businesses identify high-risk customers and take proactive actions to improve retention.


# Dataset
## The dataset contains customer details including:
### Demographics (gender, senior citizen, partner, dependents)
### Services used (internet, streaming, tech support, etc.)
### Account information (tenure, contract, billing, charges)
### Target variable: Churn (Yes / No)
